Tips

How to convert .imagma to .imagma.html

  1. Rename *.imagma to *.imagma.html.
  2. Open and then save the file on VS Code.

Rename the file in the same way when converting .imagma.html to .imagma.

How to convert .imagma or .imagma.html to .html

  1. Open the .imagma/.imagma.html file.
  2. Run the command extension.magmaNotebook.export (Click the "Export this notebook contents" button (with icon ) on the editor title menu bar or search for the command using the command palette.)
  3. Export the file with "Save as type" HTML.

The exported .html file will be smaller than .imagma.html.

How to execute codes in .imagma on the production version

  1. Click "Preview the execution contents" button (with icon ) in the notebook cell title menu bar (the upper right corner of the target cell). Then an untitled magma file will be created.
  2. Save the created file. This file is executable on the production version.
